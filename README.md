# Gerencia-Freelancer

Esse projeto está sendo desenvolvido a partir do desafio técnico proposto nesse [anúncio](https://codante.io/testes-tecnicos/teste-tecnico-entria) da empresa Entria.
Meu objetivo é desenvolver uma aplicação robusta Fullstack, seguindo as restrições impostas no desafio. Vou desenvolver uma aplicação Web que possibilite freelancers gerenciarem clientes, projetos e finanças, com foco em controle de horas e faturamento. Inicialmente não pretendo dar continuidade no projeto depois de finalizado.
